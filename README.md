# CornOS
Learning OS development

This repository is dedicated to learning how to develop and operating system by following the book [The little book about OS development](https://littleosbook.github.io/) and also with massive helps from the [OSDev wiki](https://wiki.osdev.org/).

# Prerequisites
There are prerequisites if you want to try this on your platform:
 - x86 CPU (64 bits is not necessary)
 - A linux machine (with graphical user interface)
 - softwares (qemu, grub, make, gcc, nasm, binutils)

# Build
With these installed, run
```
make run
```
To build the OS and run it with qemu

This OS may grow into a custom OS in the future depending on personal interest and laziness.
