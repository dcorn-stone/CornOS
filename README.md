# CornOS
Learning OS development

This repository is dedicated to learning how to develop and operating system by following the book [The little book about OS development](https://littleosbook.github.io/).

There are prerequisites if you want to try this on your platform:
 - x86 CPU (64 bits is not necessary)
 - A linux machine (with graphical user interface)
 - softwares (qemu, grub, make, gcc, nasm, binutils)

This OS may grow into a custom OS in the future depending on personal interest and laziness.
